# Community
Community 是牛客网讨论区实战项目，用于巩固 Web 开发所学的知识。
## 架构

- 编程语言：Java 1.8 + TypeScript
- 前端框架： Vue 3.2 + Pinia
- UI 组件：Element Plus
- 后端框架：Spring Boot 2.5.6 
- 持久层架构：MyBatis-Plus 3.3.2 + MySQL 5.7 + Redis 5.0
- 安全框架：Spring Security
